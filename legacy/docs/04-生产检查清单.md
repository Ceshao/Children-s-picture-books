# 生产检查清单（交付前逐项过）

> 这是 `00 设计标准` 第 8 节红线的"可勾选版"。每本书交付前从头过一遍，全打勾才算合格。
> 任何一项打叉 → 回到对应文档修。

---

## A. 文字（对照 00 §4、§8.1）
- [ ] 每页 1 句（最多 2 句短句）
- [ ] 每句 ≤8 词，核心句 4–6 词
- [ ] 全书总字数 150–400 词
- [ ] 有 1 句副歌，全书重复 ≥3 次，位置固定
- [ ] 全书 ≥3 处拟声/声音词
- [ ] 关键动作/情绪用**对白**，不是旁白复述画面
- [ ] 默认具体字面语言；隐喻/拟人 ≤1 处且有画面支撑
- [ ] 没有强行押韵（要么全程顺口，要么不押）
- [ ] **朗读测试**：每页大声读 ≥3 遍都顺口、副歌好背

## B. 画面（对照 00 §5、§8.2）
- [ ] 景别多样：≥1 建立镜头 + ≥1 细节/空镜 + ≥1 特写 + ≥1 满版
- [ ] 非人物页（环境/空镜/细节）≥25%
- [ ] 每页图文互补，文字不复述画面
- [ ] 高潮 ≥1 张满版大图
- [ ] 主角朝向/构图配合从左到右翻页动量

## C. 结构（对照 00 §2、§8.3）
- [ ] 10–14 个画面，起承转合齐全
- [ ] 场景之间有可见的物理过渡，无跳脱
- [ ] 结尾回到安全/温暖，情绪有落点，不说教
- [ ] 善用了封面/扉页等 peritext

## D. 一致性（对照 00 §6、§8.4）
- [ ] **全书每页尺寸完全一致**（同一画布/默认竖版 2:3，封面也一样；满版页没换横版）
- [ ] 每个角色 3–7 个身份锚点，写进了每条 prompt
- [ ] 固定主色板 + 风格描述词
- [ ] 首页 vs 末页并排比对，无角色漂移
- [ ] 全书画风/光线/色调统一

## E. show-and-tell 适配（本项目专属）
- [ ] 3 岁能靠"记副歌 + 看图"复述出大意
- [ ] 句式可预测（孩子能猜下一句）
- [ ] 有可表演的声音词/对白（孩子能用嘴和身体演）

---

### 快速反面自查（对照 00 §9）
- [ ] 没有"旁白复述画面"的句子
- [ ] 没有"每页都是主角正面中景"
- [ ] 没有成人式抽象诗意（whispered / held its breath 之类）
- [ ] 没有场景跳脱
- [ ] 没有角色漂移
