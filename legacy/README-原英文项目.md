# legacy/ —— 英文期项目归档（已退役为参考资料）

这里保存的是本工程**第一阶段**的全部成果：一套面向 **3 岁孩子 show-and-tell 上台表演/背诵** 的**英文**绘本定制工作流与 Skill（`english-picture-book` v1.4.0）。

2026-06 起，工程**彻底重构方向**：聚焦**中文幼儿园儿童（3–6 岁）自主翻阅**的绘本，以《3—6 岁儿童学习与发展指南》为政策基座，由多 Agent 团队分工创作。新主线在仓库根目录的 `skills/chinese-picture-book/` 与 `.claude/agents/`。

## 为什么保留而非删除

英文期沉淀了一批**经过验证、且与语言无关**的设计 DNA，新中文 Skill 是在其基础上改编而来，保留原件便于对照与回溯：

- **图文关系 / 翻页戏剧 / 镜头多样性 / 反面清单** —— 见 `docs/00-绘本设计标准与原则.md`
- **质量北极星 + 体感检查** —— 见 `docs/什么是好-3岁绘本北极星.md`
- **配图一致性工程**（身份锚点、固定风格块、照片+主参考图、首末页漂移检查、统一 2:3 画布）—— 见 `skills/english-picture-book/assets/image-prompt-template.md`，这是「皇冠明珠」，几乎原样移植到了新 Skill。
- **风格库**（8 种画风 + STYLE/PALETTE 块）—— 见 `docs/风格库-Style-Library.md`
- **完整工作样例 Orion** —— 见 `samples/Orion/` 与早期出图草稿 `V1/ V2/ V3/`

## 与新中文工程的关键差异

| 维度 | 英文期（本归档） | 中文期（新主线） |
|---|---|---|
| 语言 / 受众 | 英文 / 双语家庭 3 岁 | 中文 / 幼儿园 3–6 岁（小/中/大班分级） |
| 核心场景 | 上台 show-and-tell 表演、背诵 | 幼儿**自主翻阅**（看图读懂） |
| 北极星 | 给孩子一个能背诵/表演的游戏 | 让孩子**只看图**就能把故事讲对 |
| 政策基座 | 无（凭设计经验） | 《3—6 岁儿童学习与发展指南》+ 早期阅读指导 |
| 生产形态 | 单 Skill 顺序工作流 | 7 个有人格的真实子 Agent 编排 + 交叉校验 |
| 演讲套件 / 听音频 | 有 | **不做**（已移出范围） |

> 本目录内容**不再维护**，仅作参考与历史留存。`V1/ V2/ V3/` 为早期出图草稿，体积较大，已在 `.gitignore` 中排除，不推送到远端。
