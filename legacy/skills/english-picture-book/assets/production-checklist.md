# 生产检查清单（交付前逐项过，全打勾才合格）

## A. 文字
- [ ] 每页 1 句（最多 2 短句）
- [ ] 每句 ≤8 词，核心句 4–6 词
- [ ] 全书 150–400 词（能背诵优先可更短）
- [ ] 副歌 1 句，全书重复 ≥3 次，位置固定
- [ ] ≥3 处拟声/声音词
- [ ] 关键动作/情绪用**对白**，不是旁白复述画面
- [ ] 默认具体字面；隐喻/拟人 ≤1 处且有画面支撑
- [ ] 没有强行押韵
- [ ] **朗读测试**：每页大声读 ≥3 遍顺口、副歌好背

## B. 画面
- [ ] 景别多样：EST + DET + CU + FULL 各 ≥1
- [ ] 非人物页（环境/空镜/细节）≥25%
- [ ] 每页图文互补，文字不复述画面
- [ ] 高潮 ≥1 张满版大图
- [ ] 主角朝向/构图配合左→右翻页动量

## C. 结构
- [ ] 10–14 画面，起承转合齐全
- [ ] 场景间有可见物理过渡，无跳脱
- [ ] 结尾回到安全/温暖，不说教
- [ ] 善用封面/扉页等 peritext

## D. 一致性（AI 生图）
- [ ] **全书每页尺寸完全一致**（同一画布/默认竖版 2:3，封面也一样；满版页没换横版）
- [ ] 每角色 3–7 身份锚点写进每条 prompt
- [ ] 固定主色板 + 风格描述词
- [ ] 每页同附"孩子照片 + 主参考图"
- [ ] 首页 vs 末页并排，无角色漂移，且像照片里的孩子

## E. show-and-tell 适配（站到全班面前演讲）
- [ ] 目标年龄能靠"记副歌 + 看图"复述大意
- [ ] 句式可预测（孩子能猜下一句）
- [ ] 有可表演的声音词/对白
- [ ] 有**开场自我介绍** + **清楚的收尾**（The end! Thank you!）
- [ ] 有 **1 处全班参与**、且由孩子主导（如带全班接副歌）
- [ ] **图=提示**：每页插画能提示该页台词（孩子忘词也救得回来）
- [ ] **举起来全班看得清**：每页有一个醒目、高对比的主体（远看不糊）
- [ ] 主角是真实的孩子、主题是娃真爱的东西（"这就是我"）
- [ ] 台词好发音（尤其副歌/开场，适合可能双语的幼儿）
- [ ] 已生成**演讲套件**（提示卡+排练计划+Q&A），整体 2–3 分钟

### 快速反面自查
- [ ] 没有旁白复述画面 / 没有每页都是主角正面 / 没有成人式抽象诗意 / 没有场景跳脱 / 没有角色漂移
