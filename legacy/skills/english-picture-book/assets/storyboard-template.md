# 分镜表 · Storyboard（填完故事圣经后填）

> 把故事拆成约 10–14 个画面，每行定：镜头 + 画面内容 + 文字。这是写文字和生图的唯一依据。

## 镜头类型速查（每页选一个，别全用 MED）
| 代码 | 镜头 | 何时用 |
|---|---|---|
| EST | 建立镜头（高角/广角环境） | 新场景开头、交代世界、喘息页 |
| WIDE | 远景 | 角色 vs 环境、旅程 |
| MED | 中景 | 角色互动、对话 |

| FULL | 满版大图（可无字） | 高潮 |
