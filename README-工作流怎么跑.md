# 定制英文儿童绘本 · 工作流

为侄子/侄女定制英文绘本（幼儿园 show-and-tell 用），从一个点子到成品的完整流程。
面向 **3–5 岁混龄**，核心句按 **3 岁**调，目标是让孩子**自己对着图复述/表演**。

---

## 这套系统有什么

| 文件 | 作用 | 什么时候用 |
|---|---|---|
| `00-绘本设计标准与原则.md` | 大脑：所有规则的依据 | 想懂"为什么"时看；第 8、9 节随时查 |
| `templates/01-故事圣经-Story-Bible.md` | 锁角色/世界/风格/副歌 | 每本书最先填 |
| `templates/02-分镜表-Storyboard.md` | 每跨页：镜头·文字·画面 | 填完故事圣经后填 |
| `templates/03-配图Prompt模板.md` | 生成风格/角色一致的图 | 分镜定稿后出图用 |
| `04-生产检查清单.md` | 交付前红线自检 | 出成品前逐项过 |

---

## 做一本新书的步骤

```
点子 → ①故事圣经 → ②分镜表 → ③写文字 → ④出图 → ⑤检查清单 → 成品PDF
```

**① 填《故事圣经》**（`templates/01`）
锁死：一句话故事、角色锚点(3–7个)、世界标志物、风格锚点、**副歌**、声音词库。
→ 这一步决定全书一致性，别跳。

**② 填《分镜表》**（`templates/02`）
把故事拆成 10–14 个画面，每行定：镜头类型 + 画面内容 + 文字。
→ 重点：镜头别全是 MED；EST/DET/CU/FULL 各 ≥1；非人物页 ≥25%。

**③ 打磨文字**
对照 `00 §4`：每页 1 句、≤8 词、副歌重复、用对白和声音词、不复述画面、不堆诗意。
→ 做**朗读测试**：大声读 ≥3 遍。

**④ 出图**（`templates/03`）
先出角色参考图 → 封面 → 内页（每 3 张回看一致性）→ 首尾并排校验。

**⑤ 过《检查清单》**（`04`）
全打勾才算合格。有叉就回对应文档修。

**⑥ 合成 PDF**
按页顺序把图 + 文字排成绘本（沿用你现在的 PDF 合并方式）。

## 一句话提醒
> 图画讲"发生了什么"，文字只点"一个声音 / 一句对白 / 一个钩子"。
> 让孩子用**嘴和身体**参与，而不是听一段旁白。
