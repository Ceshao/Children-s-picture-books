# 合规要点 · 小熊找朋友（中班 4–5）

> Stage1 政策预简报 · 育儿顾问·苗老师产出。
> 本文是发给**作家**、**画面叙事设计师**的前置指引——动笔/分镜前先吃透。
> 题材：豆豆害羞慢一步 → 鼓起勇气分享野果 → 交到朋友。结构：一个小问题→解决（中班单线闭环）。
> 依据：《3—6 岁儿童学习与发展指南》语言领域 + `年龄分级内容规范.md` + `绘本内容红线.md` + `题材边界-健康社会科学.md`。
> 这不是裁定（裁定在校验门 A/C 出），是**目标清单 + 硬约束 + 本题材专项避雷**。

## 一、本书必须命中的《指南》目标子集（中班语言/阅读理解）

> 命中 = 不只是「没违反」，而是要**主动设计去达成**。校验门 A/C 会回头核对这几条是否真的落地。

| 目标 | 《指南》原文（4–5 岁） | 本书怎么落地 |
|---|---|---|
| **阅读理解 目标2-②（核心）** | 「能根据连续画面提供的信息，大致说出故事情节。」 | **全书脊梁**：仅凭画面，孩子要能粗略复述「豆豆想玩→不好意思→采到野果→鼓起勇气分享→交到朋友」整条线。每一页的表情/动作必须是可读的因果链，不能靠文字补叙。 |
| **阅读理解 目标2-①** | 「能大体讲出所听故事的主要内容。」 | 开始-经过-结尾清楚，主线单一，听完一遍能复述大意。 |
| **阅读理解 目标2-③** | 「能随着作品情节的展开，产生喜悦、担忧等相应的情绪反应。」 | 设计**情绪曲线**：豆豆的「想加入又退缩」要让孩子替它着急/担忧，分享成功要让孩子替它高兴。表情画到位是关键。 |
| **倾听与表达 目标2-③④（4–5）** | 「能基本完整地讲述所见所闻」「讲话比较连贯」。 | 画面要支持孩子**连贯复述**——前后页有清楚的承接（先…后…），不跳步、不留逻辑断点。 |
| **喜欢阅读 目标1-①②** | 「反复看自己喜欢的图书」「喜欢和别人谈论故事中有趣的情节」。 | 留**可回味/可讨论**的趣味点（如每次"差一点点"的小重复、野果篮的小细节），支持反复翻看与亲子/同伴讨论。 |

> 自主翻阅检验线（中班）：**遮住文字，顺着画面线索能粗略复述全程**。这是 A/C 门和童读理解门 D 的硬尺子——画面叙事设计师请以此为第一标准。

## 二、本题材必须规避的红线（害羞 / 分享 / 交友 专项）

> 这三个主题最容易写歪成「说教」和「强迫分享」。逐条避雷。

1. **【红线7 不说教 · 本书头号风险】** 主题（迈一步+分享=交朋友）**只能靠故事和画面自然浮现**，禁止任何角色或旁白说出大道理。
   - ✗ 禁止出现「要学会分享哦」「主动一点就能交到朋友」「分享让人快乐」这类训诫句/总结句。
   - ✗ 禁止贴标签式句子（「豆豆是个害羞的孩子」「豆豆变勇敢了」当结论喊出来）。
   - ✓ 用豆豆的**表情、动作、和同伴互动的结果**让孩子自己读出道理。结尾给画面（一起玩的暖场景），不给说教。

2. **【红线6 分享要自然、不强迫 · 本书第二风险】** 分享必须是豆豆**自己想分享**、出于善意与渴望加入，**不是被催、被要求、被纠正**。
   - ✗ 禁止任何角色对豆豆说「你应该分给大家」「不分享不是好孩子」；禁止豆豆因被批评/羞愧才分。
   - ✗ 禁止把"不分享"画成错误、把分享画成"交换朋友的条件"（不要写成"分了才有人理它"的功利逻辑）。
   - ✓ 分享是豆豆鼓起勇气、主动伸出手的**温暖一步**；朋友的回应是自然的开心，不是"奖励"。

3. **【红线2 不渲染负面情绪 / 红线1-4 结尾回安心】** 害羞、没能加入的"小紧张"要**很快化解**，不可放大成孤独、被排挤、被嘲笑。
   - ✗ 同伴**不可**表现为排斥、嘲笑、故意不带豆豆玩（那会变成欺凌/受伤，触红线2与价值观）。"差一步"是豆豆自己的犹豫，不是别人拒绝他。
   - ✓ 结尾必须温暖落地：豆豆被自然接纳、一起玩，回到安心。

4. **【红线5 无刻板印象】** 害羞 ≠ 缺点。不把"害羞/慢半拍"画成需要被矫正的毛病；不附加性别/外貌刻板（如"男孩不许害羞"）。豆豆的勇气是成长，不是"改正错误"。

5. **【红线9 不超龄 / 红线10 不为识字】** 见第三节硬约束。不放拼音、不做识字训练、不写脱离画面的隐喻或复杂因果句。

6. **【红线零·法律 & 红线1 安全】** 本题材天然安全（草地、野果、交友），但仍须确认：野果不暗示"可随便采食野外果子吃"的危险示范——可处理为分享/递给同伴的善意道具，画面不强调"吃下去"，避免低龄模仿采食野生植物。结尾无任何恐怖/惊吓元素。

## 三、中班硬约束摘要（文字 / 句式 / 画面 / 结构）

> 作家管文字与结构，画面叙事设计师管画面与翻页。逐项当契约执行。

### 文字（天花板，不是目标）
- **每页 1 句，≤10 字**；能更少更好，可有无字页让画面独立叙事。
- 文字只**补画面之不足**（一句心声、一个声音、一个翻页钩子），**绝不复述画面已画出的内容**。
- 朗读测试：每页连读 3 遍顺口，无拗口、无生僻字、无拼音。

### 句式与词汇
- **简单句 + 轻并列**（「……然后……」）；**禁用**复杂因果/假设句（「因为…所以…」「如果…就…」留给大班）。
- 词汇用具体当下动作；可引入**画面能解释的新词**（草地、野果、一篮——画面画了才可出现）。
- **无新颖隐喻、无抽象概念**；不以识字为目的。

### 画面复杂度
- 主体**豆豆醒目** + **一个清楚的次要元素/简单场景**（小伙伴、草地、野果篮），**仍不杂乱**。
- 关键：**表情与动作要读出因果**——豆豆的犹豫→鼓起勇气→分享→大家开心，逐页可见、可连成线。
- 温暖、有童趣、幼儿视角；色彩可鲜明生动；不要"精致但无童趣"的成人审美（红线11）。

### 叙事结构与翻页
- **线性单线，一个小问题→解决**；开始-经过-结尾清楚。
- 小紧张（没能加入、害羞）**很快化解**，结尾温暖回到安心。
- **翻页钩子**：用"重复但情境每次变化"——每次"差一点点"，制造"然后呢？"的期待；到最后"这次不一样了"完成闭环。重复**不可原样复读**，每次情境要有变化。

### 篇幅
- **10–12 个画面**（封面另计）；项目卡建议锁 11 画面，与中班区间一致。

## 四、给两位创作者的一句话提醒

- **作家**：你最大的敌人是"说教句"和"强迫分享"。把道理交给画面，文字只留钩子和心声；≤10 字、简单句，到此为止。
- **画面叙事设计师**：你的成败标准只有一条——**遮住文字，孩子能不能顺着画面复述全程并替豆豆着急/高兴**。表情和"差一步→迈出去"的动作因果，是这本书的命脉。
