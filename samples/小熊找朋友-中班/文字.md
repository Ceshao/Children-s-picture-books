# 每页文字 · 小熊找朋友

> 作家(林禾) Stage4 产出，校对(一字) 校验门 B 回写。年龄段：**中班(4–5)**　每页字数上限：**≤10 字、1 句**。
> 原则：**只补画面给不了的**（一个声音/一句对白/一个翻页钩子），**绝不复述画面已画出的东西**。能更少更好，可留白无字。

| 页码 | 最终文字（≤10 字；可留空=无字） | 类型（对白/拟声/旁补/重复句/无字） |
|---|---|---|
| 封面 | 小熊找朋友 | 书名 |
| 1 | 好想一起玩。 | 旁补（补画面给不了的「心里的想」；远景画面已画出草地与小伙伴，不复述） |
| 2 | 豆豆，又慢一步。 | 重复句①（v2：重复句情境变体—「又」承接P1初次想加入，犹豫藏手） |
| 3 | 豆豆，慢了一步。 | 重复句②（v2：重复句情境变体—保留骨架基准句，正对「追蝴蝶错过」） |
| 4 | 豆豆，还是慢一步。 | 重复句③（v2：重复句情境变体—「还是」叠加前两页，推到情绪最低点「想喊没敢」） |
| 5 | 咕噜——一颗果子。 | 拟声＋旁补（果子滚动的一声；换气页，轻轻把目光引到篮子，画面已是篮子静物） |
| 6 | 呼—— | 拟声（深吸一口气；画面已画抬头与决心，只补这口气的声音）｜校对(一字)删去破折号后多余句号 |

## 自检
- [x] 每页 ≤10 字、1 句；P7/P10 留白无字，能更短就更短
- [x] 无超龄词、无脱离画面的隐喻、无以识字为目的的设计
- [x] 没有一页的文字在复述它的画面（P1 补「想」、P5/P6 补声音、P9 补对白，均为画面给不了的）
- [x] 重复句 v2 情境变体：固定节拍 P2/P3/P4 同句式骨架「豆豆，____一步。」（又慢／慢了／还是慢），措辞随情境微变、仍同一重复感（中班「重复但情境每次变化」）；反转句「这一次，豆豆没有等。」落在 P8
- [x] 每页默念 3 遍顺口；首尾「好想一起玩」↔「现在，一起玩」轻轻收环
