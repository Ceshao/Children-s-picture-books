# 出图与排版说明 · 小熊找朋友（中班 4–5）

> 主编（墨白）Stage6 产出。这是**给人执行的简明清单**——照着做就能把这本书从「prompt」变成「可印的 PDF」。
> 所有 prompt 在 `配图Prompt.md`，逐页文字在 `文字.md`，总装对照在 `成书包.md`。
> 画风：古典水彩（#1）。统一画布：竖版 2:3（≈1024×1536），**全书同尺寸，封面也是，满版页不换横版**。

---

## 总览：六步走

```
① 主参考图  →  ② 封面  →  ③ 按页出图(每3张查一致性)  →  ④ 首末页校验  →  ⑤ 统一裁切  →  ⑥ 排版PDF
```

---

## ① 先出「角色参考图」——全书的命根子

> 本书没有真实孩子照片，主参考图就是一致性的唯一锚。**先把它出到满意，再动任何内页。**

1. 打开 `配图Prompt.md` →「一、角色参考图流程」。
2. 复制 **A. 豆豆主参考图 prompt**，生成 3–4 张。
3. 对照 6 条锚点挑一张最准的，**存为「主参考图」**：圆头圆耳 / 蜜糖棕绒毛 / 仅奶黄背带裤无鞋 / 草绿围巾 / 两点脸颊红晕 / 害羞低视线；且「害羞 vs 勇气」两组对照动作清楚。
4.（建议）复制 **B. 配角三只参考图 prompt**，出一张三只合照，**存为「配角参考图」**（供 P8/P9 多人页认得出同三只）。
5. 之后**每一页出图都附上主参考图**；多人页（P8/P9/P10/P11）再附配角参考图。

---

## ② 出封面——最先暴露风格问题

1. 到 `配图Prompt.md` →「封面 · COVER」，整段复制 prompt，附主参考图，出图。
2. 这张最能看出画风对不对：暖光、水彩质感、篮子是否最亮最抢眼。**封面满意 = 全书风格基调定了**，不满意就在这里调，别带着风格问题往下走。

---

## ③ 按 P1→P11 顺序出内页——每 3 张回看一次一致性

1. 严格按 **P1 → P11** 顺序，每页到 `配图Prompt.md` 对应小节整段复制 prompt，附主参考图（多人页加配角参考图）。
2. **每出满 3 张就停下回看一次**，用 `配图Prompt.md` →「四、一致性校验清单」逐条核：
   - **B. 豆豆锚点**：圆头圆耳/蜜糖棕/仅奶黄背带裤无鞋/草绿围巾/两点红晕；害羞页低视线、勇气页(P6–P8)抬头踮脚双手前递。
   - **C. 配角**：始终恰好 3 只、认得出、全程友好无皱眉嘲笑；P1 追蝴蝶、P3 结伴跑开**不撞图**。
   - **D. 篮子**：每页找得到、是最亮点；满→空弧线对（封面满 / P8 递出 / P10 空大半 / P11 空篮搁草地）；P3 一颗滚落、P5 一颗到篮边。
   - **E. 风格/光线**：古典水彩、暖光、同一色板（ambers/sage/sky blue/cream）不忽明忽暗；无残留文字。
3. **漂移就当场修**，别将就：重附主参考图 / 加重锚点描述 / 局部重绘（inpainting）。
4. 三个最易翻车处重点盯：
   - **P3**：半空的手 + 抬着未落的脚 + 正滚落的果子三处落空**同框且在前景**；小伙伴只剩较小背影、明确「在跑离」。
   - **对照动作**：P2 藏手 ↔ P8 前递、P4 低头 ↔ P6 抬头，要画出可见反差。
   - **满版 P8**：保持同一 2:3 画布、靠构图铺满，**不换横版**；所有视线锁向篮子。

---

## ④ 首末页校验——P1 vs P11 并排

1. 全部出完后，把 **P1 和 P11 并排**比对（这是最考验一致性的两张：同一角草地）。
2. 核对：是否同一片草地同一角、同一只篮子、同一只豆豆；豆豆从 **P1 左下角落单 → P11 人群正中央**的位置对照是否读得出。
3. 不一致就重绘 P11（以 P1 为基准对齐场景）。

---

## ⑤ 统一裁切——排版前对齐尺寸

1. 工具出图尺寸常有细微出入。排版成 PDF 前，把**全部图（含封面、满版 P8）统一裁成同一 2:3**。
2. 满版 P8 仍是 2:3，**不要**因为「满版」就裁成横版。
3. 裁切后再扫一眼：构图主体（豆豆/篮子）没被裁掉、关键细节（P3 三处落空、P10 众手）仍完整在画面内。

---

## ⑥ 排版成 PDF——每页图下居中放文字

1. 每页一张图，**图下居中**放该页文字（文字见 `文字.md` / `成书包.md` 总装表）。
2. 文字逐页（直接照抄，**别改字**）：
   - 封面：小熊找朋友
   - P1：好想一起玩。
   - P2：豆豆，又慢一步。
   - P3：豆豆，慢了一步。
   - P4：豆豆，还是慢一步。
   - P5：咕噜——一颗果子。
   - P6：呼——
   - P7：（无字，留白）
   - P8：这一次，豆豆没有等。
   - P9：「一起玩吧！」
   - P10：（无字，留白）
   - P11：现在，一起玩。
3. 字体建议：圆润、清晰、够大的中文字体（幼儿读物风格）；**不放拼音**。
4. P7/P10 **保持留白无字**，别硬塞文字。
5. 全书排完通读一遍：图文是否互补（文字没在复述画面）、留白页是否舒服、首尾「好想一起玩」↔「现在，一起玩」是否收得上。
