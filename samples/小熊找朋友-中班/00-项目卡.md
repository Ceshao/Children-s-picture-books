# 项目卡 · 小熊找朋友（暂定名 / 可终审时定稿）

> 主编（墨白）Stage0 产出。这是要**与用户确认**的第一份产物，也是全程的设定与决策档案。
> 本书定位：**样例绘本**（虚构角色、无真实孩子照片），用于演示「中班(4–5)」绘本的完整工作流与成片质感。

## 基本设定
- 孩子姓名 / 称呼：（样例书，无真实孩子）主角为虚构小熊「**豆豆**」
- 年龄：**4 岁半** → **年龄段：☐ 小班(3–4) ☑ 中班(4–5) ☐ 大班(5–6)**
- 是否有真实孩子照片：☐ 有（走相似度流程）　**☑ 无（虚构角色，跳过相似度门）**
- 题材种子 / 主题 / 场合：
  豆豆很想和草地上的小伙伴们一起玩，但每次都有点**不好意思、慢一步**，没能加入。
  后来它发现自己采的**一篮野果**可以分享，于是**鼓起勇气**分给大家，交到了朋友。
  - 主题落点：**主动迈出一步 + 分享 = 交到朋友**（社会性/情绪：害羞→鼓起勇气）。
  - 结构定性：中班标准「**一个小问题 → 解决**」单线闭环。
- 画风意向（见 `../../skills/chinese-picture-book/references/风格库.md`）：
  ☐ 待试片　**☑ 初选：古典水彩（1 号 · Classic Watercolor，默认推荐）**
  - 选型理由：温暖、低龄友好度高、质感好，契合"温柔交友/草地自然"题材；**最终以风格试片定稿**（建议出 1 张试片确认）。
- 篇幅：约 **10–12** 个画面（封面另计）

## 本年龄段关键约束（摘自 `../../skills/chinese-picture-book/knowledge/年龄分级内容规范.md`）

> 发展主线（中班 4–5）：**「能根据画面线索（表情、动作）理解主要情节」** → 画面要给出**可读的因果线索**，孩子能粗略复述全程。

- **每页文字上限**：1 句，**≤10 字**（字数是天花板，能更少更好；文字只补画面之不足，绝不复述画面已画出的内容）。
- **页数区间**：**10–12** 个画面（与本书篇幅一致）。
- **句式**：简单句 + 轻并列（「……然后……」）；用具体当下动作；可引入「画面能解释」的新词（如"草地""野果""一篮"）；**不放拼音、不以识字为目的、无脱离画面的隐喻**。
- **叙事结构**：☐ 重复/累积环(小班)　**☑ 一个小问题→解决(中班)**　☐ 含1–2转折(大班)
  - 要求：开始-经过-结尾清楚；小紧张（没能加入、害羞）**很快化解**；结尾温暖回到安心（交到朋友）。
- **画面复杂度**：主体（豆豆）醒目 + **一个清楚的次要元素 / 简单场景**（小伙伴、草地、野果篮），仍**不杂乱**；关键是**表情与动作要能读出因果**（豆豆的犹豫→鼓起勇气→分享→大家开心）。
- **翻页钩子**：重复但情境每次变化，制造明确期待（如每次"差一点点"，到最后"这次不一样了"）。
