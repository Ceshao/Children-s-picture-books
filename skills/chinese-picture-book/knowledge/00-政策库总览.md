# 政策知识库 · 总览与使用说明

本目录是整套中文幼儿园绘本工作流的**政策与发展基座**。所有内容创作（故事、画面、文字）都必须以此为准绳。它把《3—6 岁儿童学习与发展指南》（教育部 2012，以下简称《指南》）+ 公开早期阅读/语言指导，蒸馏成**按需加载的小文件**，让每个 Agent 只读自己需要的那一片，既不超龄、也不漏掉发展依据。

> 核心立场（来自《指南》语言领域）：**「用机械记忆和强化训练的方式让幼儿过早识字，不符合其学习特点和接受能力。」** 因此本工程的绘本以**画面叙事**为脊梁、文字极简、面向**幼儿自主翻阅**——孩子大多不识字，靠**看图**读懂故事。这与《指南》阅读与书写准备 目标2「会看画面，能根据画面说出图中有谁、发生了什么事」完全一致。

## 一、矩阵地图（《指南》原生结构）

《指南》是 **5 领域 × 3 年龄段** 的「目标 + 教育建议」矩阵：

| 领域 | 子领域 | 对绘本的主要作用 |
|---|---|---|
| 健康 | 身心状况 / 动作发展 / 生活习惯与生活能力 | 题材边界（安全示范、生活自理）、情绪基调 |
| **语言** | **倾听与表达 / 阅读与书写准备** | **最吃重**：直接决定文字量、句式、阅读理解的分级目标 |
| 社会 | 人际交往 / 社会适应 | 题材与价值观（友爱、规则、归属感）、内容红线 |
| 科学 | 科学探究 / 数学认知 | 题材（自然、因果、数量）、画面可观察性 |
| 艺术 | 感受与欣赏 / 表现与创造 | 画面审美取向：不以成人标准评判、不追求"完美"、保护想象 |

三个年龄段对应幼儿园：**小班 3–4 岁、中班 4–5 岁、大班 5–6 岁**。

## 二、文件结构与各 Agent 的加载映射

```
knowledge/
├── 00-政策库总览.md            ← 本文（人/Agent 入口，先读这里）
├── 绘本内容红线.md             ← 【契约】内容安全/价值观红线 + 分龄禁忌
├── 年龄分级内容规范.md         ← 【契约】分段 文字量/句式/词汇/页数/画面复杂度 量表
├── 自主翻阅设计原则.md         ← 画面即叙事原则（看图能讲对故事）
├── 指南/
│   ├── 语言/{小班-3-4,中班-4-5,大班-5-6}.md   ← 语言目标的绘本化解读（最常用）
│   └── 艺术/{小班-3-4,中班-4-5,大班-5-6}.md   ← 画面审美的发展依据
│   └── 题材边界-健康社会科学.md                ← 三域对「题材是否适龄」的指引
├── 原文摘录/语言-原文.md, 艺术-原文.md        ← 《指南》逐字原文，供合规裁定引用
└── web-synthesis/{早期阅读与图画书-综述.md, 来源清单.md}   ← 公开权威指导 + 出处
```

**两份「契约」文件**（`绘本内容红线.md` + `年龄分级内容规范.md`）是每一次合规裁定都要引用的依据——任何产物都对照它逐项过门。

### 各 Agent 加载哪些文件（控上下文）

| Agent | 加载 |
|---|---|
| 育儿顾问（把关者，唯一加载全套门） | `绘本内容红线.md` + `年龄分级内容规范.md` + `指南/语言/<本段>.md` + `原文摘录/*`（引用用）+ `指南/题材边界-健康社会科学.md` |
| 作家 | `年龄分级内容规范.md`（本段的文字/句式/词汇行）+ `指南/语言/<本段>.md` + `自主翻阅设计原则.md` |
| 画面叙事设计师 | `自主翻阅设计原则.md` + `指南/艺术/<本段>.md` + `年龄分级内容规范.md`（画面复杂度行） |
| 校对 | `年龄分级内容规范.md`（词汇/句式行）+ `指南/语言/<本段>.md`（阅读与书写准备） |
| 童读小测者 | **只**读 `自主翻阅设计原则.md` 末尾的「童读协议」+ 本段阅读理解预期——保持「天真」，不读创作技法，以免被技法说服 |
| 绘画师 | 一般不需政策库；如涉及题材合规，读 `绘本内容红线.md` |
| 主编 | `绘本内容红线.md` + `年龄分级内容规范.md`（终检对照） |

## 三、引用规范

- 育儿顾问出具 `合规裁定.md` 时，**必须**引用具体依据：要么引 `原文摘录/*` 的《指南》原文，要么引 `年龄分级内容规范.md` / `绘本内容红线.md` 的条目编号。
- 量表中标「经验区间」的数字是行业经验，**非《指南》硬性规定**，作为操作参考；标「《指南》」的是有官方依据的。

## 四、维护

- 《指南》原文以 `原文摘录/*` 为准（从仓库根目录 PDF 转录）。
- 网络综述与数字区间随认知更新，更新时同步 `web-synthesis/来源清单.md`。
- 修改契约文件后，应回归跑一遍样例，确认裁定门与童读门仍按预期工作。
