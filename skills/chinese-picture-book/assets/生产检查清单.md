# 生产检查清单（终检 · 主编出书前逐项过）

> 主编(墨白) Stage6 终检使用。任一项不过 → 不出书，回炉。

## A. 文字（按年龄段）
- [ ] 每页 ≤ 本段字数上限（小班≤6 / 中班≤10 / 大班合计≤16）；能更短就更短
- [ ] 无超龄词、无脱离画面的隐喻、无以识字为目的的设计
- [ ] 重复句一致、在固定节拍；拟声词自然好念
- [ ] 每页朗读 3 遍顺口

## B. 画面叙事（自主翻阅核心）
- [ ] **仅凭画面可还原故事**（童读小测者已判「通过」）
- [ ] 没有一页文字在复述其画面（图文互补，非复读）
- [ ] 镜头有变化：远景/特写/细节/满版 各 ≥1；非人物页 ≥约25%
- [ ] 高潮为一个满版页；相邻页有可见连接线索（无场景跳变）

## C. 结构与情绪
- [ ] 结构符合本段（小班重复环 / 中班单问题闭环 / 大班含转折）
- [ ] 页数在本段区间
- [ ] 结尾回到安心、温暖

## D. 一致性（出图工程）
- [ ] 角色身份锚点 + 风格块每页逐字一致
- [ ] 每页同一画布尺寸（封面也是）；满版页未换横版
- [ ] 配图 prompt 与画面脚本叙事节拍匹配；附「照片+主参考图」流程已说明

## E. 政策门（必须全 PASS）
- [ ] 内容红线：法律(未保法/出版条例)、安全、价值观、适宜性 均无触线
- [ ] 年龄段达标：育儿顾问校验 A、C 均 PASS
- [ ] 理解门：童读小测者校验 D 判「通过」
- [ ] 题材正向、贴近经验、无刻板与说教

## F. 交付物齐备
- [ ] 成书包.md（按页：画面要点 + 文字 + 配图prompt链接）
- [ ] 出图与排版说明.md
- [ ] 项目卡决策日志已记录各门结论
